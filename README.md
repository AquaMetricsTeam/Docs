# AquaMetrics

AI-Powered Collaborative Decision Support Platform for Competitive Swimming Academies.

## Project Overview

AquaMetrics is an intelligent platform designed to assist swimming academies by combining AI with sports science to support coaches, athletes, and administrators in making better training decisions.

The system aims to:

- Generate personalized training recommendations.
- Assist coaches with performance analysis.
- Organize academy operations.
- Centralize athlete information.
- Support evidence-based coaching using AI and RAG.
